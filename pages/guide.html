📖 小白部署教程

零基础 · 10分钟上线 · 完全免费

1 注册 GitHub 账号

访问 github.com 注册一个免费账号，这是存放我们网站代码的地方。

💡 使用邮箱注册，验证邮箱即可。用户名建议用英文，如 charm-tools

2 下载/复制代码

把整个 charm-toolbox 文件夹下载到你的电脑上，或者直接使用下面命令：

# 克隆仓库到本地（如果你会用Git的话）
git clone https://github.com/你的用户名/charm-toolbox.git

# 或者直接下载ZIP压缩包
# 然后解压到电脑上

📁 关键文件说明：

charm-toolbox/
├── index.html          # 主页（最重要的文件）
├── css/
│   └── style.css       # 样式文件
├── js/
│   └── data.js         # 工具数据（可以自己修改）
├── pages/
│   ├── guide.html      # 本教程页面
│   ├── money.html      # 赚钱攻略
│   ├── blog.html       # 博客
│   └── links.html      # 友情链接
└── images/             # 放图片的文件夹

3 创建 GitHub 仓库

登录 GitHub 后：

  1. 点击右上角 + 号 → New repository
  2. 仓库名输入 charm-toolbox
  3. 选择 Public（公开）
  4. 点击 Create repository

4 上传代码到 GitHub

方法一（推荐小白）： 直接在 GitHub 网页上传

  1. 在新创建的仓库页面，点击 Add file → Upload files
  2. 把电脑上解压后的所有文件和文件夹拖进去
  3. 滚动到底部，点击 Commit changes

方法二（命令行）：

cd charm-toolbox
git init
git add .
git commit -m "🎉 初始化魅力工具箱"
git branch -M main
git remote add origin https://github.com/你的用户名/charm-toolbox.git
git push -u origin main

5 注册 Cloudflare 并部署

访问 dash.cloudflare.com 注册免费账号。

  1. 登录后，左侧菜单点击 Workers & Pages
  2. 点击 Pages → Connect to Git
  3. 授权 GitHub，选择 charm-toolbox 仓库
  4. Framework preset 选择 None
  5. Build command 留空
  6. Build output directory 输入 /
  7. 点击 Deploy，等待约1分钟

🎉 部署完成！你会得到一个 https://xxxx.pages.dev 的网址，直接访问即可！

6 绑定自定义域名（可选）

  1. 在 Cloudflare Pages 项目页 → Custom domains → Set up a custom domain
  2. 输入你的域名（如 charm-toolbox.com）
  3. 推荐在 Namecheap 或 Dynadot 购买域名，首年约¥20-50
  4. 也可以使用 免费域名：nic.ua（免费的 .pp.ua 域名）

7 开启 HTTPS

Cloudflare 自动提供免费的 SSL 证书！无需手动配置，部署后自动开启 HTTPS，你的网站就是 https:// 安全连接。

8 自定义修改网站

想修改网站内容？超简单：

  • 修改工具名称/描述 → 编辑 js/data.js 文件
  • 修改网站标题/颜色 → 编辑 css/style.css 文件的 :root 变量
  • 修改页面文字 → 编辑对应的 .html 文件
  • 添加图片 → 放到 images/ 文件夹
  • 修改友情链接 → 编辑 pages/links.html

改完后，在 GitHub 仓库上传新文件，Cloudflare 会自动重新部署（约1-2分钟生效）！

9 接入统计工具

了解有多少人访问你的网站：

  • 51la (免费，国内速度快) → 51.la
  • Google Analytics (国际标准) → analytics.google.com
  • Umami (开源轻量) → umami.is

在 index.html 底部找到统计代码，替换成你的即可。
